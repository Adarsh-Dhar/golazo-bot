# GolazoChat

A Telegram bot that narrates live World Cup matches (odds swings + score
events, via TxLINE) and answers in-chat questions about the bound match.

## Project layout

```
src/
  index.ts            # entry point — wires bot + TxLINE streams together
  config.ts            # env loading, network endpoints
  bot/
    commands.ts        # /start, /help, /match, /ask
    qa.ts               # question answering over match state (+ honest gaps)
    narrator.ts         # odds-swing detection -> push alerts
  txline/
    auth.ts             # guest JWT + free-tier activation
    client.ts            # REST wrapper (fixtures/odds/score)
    stream.ts            # SSE consumer
    test-auth.ts         # standalone auth smoke test
    test-stream.ts        # standalone stream smoke test
  state/
    matchStore.ts        # SQLite: match state, odds ticks, events, chat bindings
dist/                   # compiled output (gitignored, generated by `npm run build`)
```

## Setup

```bash
npm install
cp .env.example .env
# fill in TELEGRAM_BOT_TOKEN and SOLANA_KEYPAIR_PATH
solana-keygen new -o solana-devnet-keypair.json   # if you don't have one yet
```

## Test TxLINE integration in isolation (before touching the bot)

```bash
npm run test:txline-auth     # confirms guest JWT + activation works
npm run test:txline-stream   # opens odds+scores SSE for 60s, logs raw payloads
```

## Run the bot

```bash
npm run dev      # tsx watch mode, no build step
# or
npm run build && npm start
```
